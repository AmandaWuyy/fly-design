# Fly Design

## 技术栈

Vue3 + TS

## 安装工具

```shell
> npm i eslint prettier-eslint eslint-config-prettier --save-dev
> npm install @vue/eslint-config-prettier
> npm install sass sass-loader -D
```

## 遇到的问题

```shell
1.全局注册的组件无法使用
解决办法：全局注册的组件用小写字母

2.textarea伪类选择器:focus不起作用
解决办法：outline: none;
```
